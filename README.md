Ardes Storage Module 

Описание:
Конзолна апликация за управление на инвентар използвайки JSON за съхранение.

Имплементирах следните функционалности:

- Add product
- Update product
- Delete product
- Search product by ID
- Search product by Name
- Filter products by quantity
- Inventory reports
  * Total Quantity
  * Total inventory value
  * Low stock report
  * Total products count

Структурата на проекта е следната:

- Data
- Models
- Repositories
- Services
- Reports

Data Storage:

- Всички продукти се съхраняват в Data/products.json

Как се използва: 

- Пуска се апликацията от Program.cs
- Демото автоматично извършва всички имплементирани функионалности и ги показва като резултат в конзолата.
